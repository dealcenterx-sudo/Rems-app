| Boundary | Description |
|----------|-------------|
| operator → Firebase Console / Vercel deploy | The operator creates indexes and confirms deployed headers in external dashboards; risk is leaking secret values into repo/docs or asserting READY when an index is still Building |
| browser → api/csp-report.js (open beacon) | The CSP report endpoint accepts unauthenticated violation reports and forwards to Sentry; documented low-risk open beacon (SEC-02) |
| deployed bundle → CSP Report-Only policy | Report-Only observes but does not block; a mis-scoped allowlist here becomes an enforcement risk only in Phase 8 |
