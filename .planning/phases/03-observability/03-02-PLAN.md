| Boundary | Description |
|----------|-------------|
| client → serverless handler | Untrusted request bodies/tokens cross here; the wrapped handler is where an uncaught throw would surface to Sentry |
| serverless function → Sentry SaaS | Server error payloads leave to `https://*.ingest.sentry.io`; must carry only method + route, not bodies/tokens |
| runtime env → function | `SENTRY_DSN` is read at runtime from `process.env`; must never be hardcoded |
